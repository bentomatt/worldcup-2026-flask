# World Cup 2026 — Flask Edition

Self-hostable World Cup 2026 app built with Flask + SQLite + Tailwind CSS CDN.

## Setup

```bash
pip install -r requirements.txt
python app.py
```

Open http://localhost:5000 — that's it, no external services needed.

## Features
- Match schedule with countdowns
- Score predictions (3pts exact / 1pt correct result)
- Group bracket competition & leaderboard
- Friend bets
- Per-match group chat
- Golden Boot pick
